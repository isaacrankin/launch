
#Installation & Usage

You will first need Node.js installed, find details here:
http://nodejs.org/

1. Open the dev directory in your terminal
2. Install JS dependencies:

	bower install

3. Install Grunt dependencies:

	npm install

4. Run Grunt

	grunt

5. Watch for changes with Grunt:

	grunt watch-all
